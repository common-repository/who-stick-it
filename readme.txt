"Who stick it" can make a sticky menu effect of any part on your website !

== Description ==

* you can add zone stikable
* you can edit this zone
* you can delete this zone
* you can make an class or id stikable or what ever you want
* you can exclude stiky on some page 


== Installation ==

1. Upload the folder `Who-stick-it` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new sticky on the option panel
